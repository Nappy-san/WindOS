Kill a process.

Parameters
process_idPID to kill.
wstatusKill status.
Returns
0 on success. -1 on failure.

When killing a process, two things can happen:

  • if the parent was waiting for his death, the process is immediately freed, and the parent is notified.
  • if the parent isn't, the process is put in zombie mode.
