Detailed Description

Initialization features before starting the kernel. Launched by loader.S, this code set up the MMU, clear caches and go into the high-adressed kernel.

init_setup_ttb ( )

Build the kernel TTB that will map the high address to kernel code and peripherals. The mapping used is:

  • 0x00000000 to 0x7fffffff: user space memory.
  • 0x80000000 to 0xbeffffff: physical translation of RAM.
  • 0xbf000000 to 0xbfffffff: peripherals mapping.
  • 0xc0000000 to 0xefffffff: kernel heap.
  • 0xf0000000 to 0xffffffff: kernel code, data and rodata.
